Swell - Mar de fondo

En este apartado configuramos el swell necesario para surfear ene ste spot. Ten en cuenta que las medidas que usen van a ser el mar de fondo.

El mar de fondo no tiene porque coincidir con el tamaño real de las olas en la costa. Lee este articulo si quieres saber mas sobre este tema.

El Periodo está directamente relacionado con la energía de las olas. Por definición el periodo es el tiempo medio (en segundos) que transcurre entre que las dos crestas de dos olas consecutivas pasar por el mismo punto en alta mar. Lee este articulo si quieres saber mas sobre este tema.
